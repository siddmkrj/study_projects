Problem Statement

ReneWind is a company that wants to stop wind turbine machines from breaking down. These turbines help make electricity using wind. If a part breaks, its expensive to replace it. But if we know early that something is about to break, we can fix it before that happens. This is cheaper and better.

To solve this, ReneWind collected information from sensors attached to wind turbines. These sensors give details like temperature, wind speed, humidity, and many other things (40 features in total). They also told us if the machine failed (1) or didnt fail (0).

Your job is to:

  • Build machine learning models using this data.
  • Predict which machines are likely to fail soon.
  • Help the company reduce repair costs by fixing machines before they break.

Data

There are two files

  1. ReneWind_Train.csv
  • Use this for training and testing different models.
  • It has 40 input columns (predictors) and 1 output column (target).
  1. ReneWind_Test.csv
  • This is for final testing of your best model.
  • It also has 40 input columns and 1 output column (target).
